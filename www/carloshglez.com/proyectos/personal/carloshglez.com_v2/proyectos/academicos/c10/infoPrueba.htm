Página de prueba


En esta página hay que poner solamente
código html para construir el contenido.

Guardar los archivos con el nombre de
"info" + "Titulo" + .htm y colocar este nombre en el arreglo
llamado Archivo() que está en funcionAjax.js

Las funciones y variables de los efectos que
tenga la página, se colocaran en un arhivo js con el nombre
de "efecto" + "Titulo" + .js y se le llamará desde
el archivo prototipo .htm con las etiquetas script.
